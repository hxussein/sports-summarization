# Telegram AI Bot

## Description
An AI chatbot built using n8n, Telegram Bot API, and OpenAI.

The bot receives messages from users on Telegram and generates intelligent responses using OpenAI.

## Tools Used
- n8n
- Telegram Bot API
- OpenAI API

## Workflow
User sends a message on Telegram → n8n receives it → OpenAI generates a response → Bot replies to the user.
