# ⚽ Automated Sports Newsletter Generator (n8n + AI + Telegram)

نظام أوتوماتيكي متكامل لمتابعة وتلخيص الأخبار الرياضية وتوليد نشرة إخبارية يومية باستخدام الـ AI وإرسالها فوراً عبر تليجرام.

---

## 🌟 مميزات المشروع (Features)

* **Fetch Sports News:** سحب أحدث الأخبار الرياضية فور حدوثها عبر مصادر الـ RSS Feed.
* **AI Summarization:** معالجة وصياغة الأخبار باستخدام **AI Agent (OpenAI)** لتلخيص أهم الأحداث في نقاط المزايا الجذابة.
* **Smart Formatting:** تنسيق النشرة بشكل احترافي مع إيموجيز مناسبة للعرض السريع على الموبايل.
* **Automated Delivery:** إرسال النشرة تلقائياً عبر بوت **Telegram** في مواعيد محدودة.

---

## 🛠️ التكنيكات والأدوات المستخدمة (Tech Stack)

* **n8n:** لبناء وتصميم الـ Workflow والربط بين الخدمة والأخرى.
* **OpenAI (GPT):** لتحليل وصياغة المحتوى الإخباري.
* **RSS Feed Read Node:** لجلب الأخبار لحظة بلحظة.
* **Telegram Bot API:** لنشر الرسائل مباشرة للمستخدمين أو القنوات.

---

## 🏗️ طريقة عمل الـ Workflow (How it Works)

1. **Schedule Trigger:** يشتغل الأوتوميشن تلقائياً حسب الجدول المخصص (مثلاً يومياً).
2. **RSS Feed Read:** يتم جلب الأخبار من رابط الـ RSS المعتمد.
3. **AI Agent:** يقرأ عناوين وتفاصيل الأخبار ويصيغ نشرة مقتضبة وممتعة باللغة العربية.
4. **Telegram Node:** تُرسل النشرة النهائية مباشرة إلى حساب/قناة Telegram.

---

## 🚀 طريقة التشغيل (Setup & Usage)

1. استورد ملف الـ Workflow (`.json`) داخل منصة **n8n**.
2. أضف مفاتيح الربط (Credentials) الخاصة بـ:
   * **OpenAI API Key**
   * **Telegram Bot Token**
3. قم بتعديل الـ `Chat ID` الخاص بحسابك في نود تليجرام.
4. فعّل الـ Workflow واستمتع بالنشرة الرياضية! ⚽🔥
